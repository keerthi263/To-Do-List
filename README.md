# To-Do-List
# To-Do List Web Application

## Overview

This is a simple To-Do List Web Application built using Python, SQL, HTML, and CSS. The application allows users to add, update, delete, and view tasks through a web browser.

## Screenshot

![Home Page](screenshots/homepage.png)
## Features

* Add new tasks
* View all tasks
* Update existing tasks
* Delete tasks
* Store tasks permanently using SQLite database
* Simple and user-friendly interface

## Technologies Used

* Python
* HTTP Server (BaseHTTPRequestHandler)
* SQL
* HTML
* CSS

## How to Run

Step 1: Create Database

Run the database creation script:
python database.py

Step 2: Start Server

Run:
python server.py

###
Step 3: Open Browser

Visit:
http://localhost:8000

## Available Operations

### Add Task

Enter a task and click Add.

### Update Task

Enter Task ID and new task name.

### Delete Task

Enter Task ID and click Delete.
